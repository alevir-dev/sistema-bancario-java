# 🏦 Sistema Bancário em Java

> Sistema de gerenciamento bancário desenvolvido em Java puro, com foco em boas práticas de orientação a objetos e organização de código em camadas.

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![Status](https://img.shields.io/badge/Status-Em%20desenvolvimento-yellow?style=for-the-badge)

---

## 🚀 Funcionalidades

- [ ] Criação de conta bancária com nome e CPF
- [ ] Depósito de valores
- [ ] Saque com validação de saldo insuficiente
- [ ] Consulta de saldo
- [ ] Extrato com histórico de movimentações

---

## 🏗️ Arquitetura

```
src/
├── main/
│    └── BancoMain.java    # Ponto de entrada e interface com usuário
└── service/
├── Conta.java  # Modelo de dados
└── BancoService.java  # Regras de negócio
```
---

## 🛠️ Tecnologias

- Java 17+

---

## ▶️ Como executar

**Pré-requisitos:** Java 17+
```bash
git clone https://github.com/alevir-dev/sistema-bancario-java.git
```

1. Abra o projeto na sua IDE (IntelliJ, Eclipse, etc)
2. Execute a classe `BancoMain`
